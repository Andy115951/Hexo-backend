# AI绘画工具从入门到精通：零基础也能画出专业级作品

你想用AI画图，但看到那些复杂的参数、模型、提示词就头大？

别担心，我手把手教你，从零开始掌握AI绘画。不需要美术基础，不需要编程知识，只要你会打字，就能画出惊艳的作品。

今天我们讲最实用的两款工具：Midjourney（简单易用）和 Stable Diffusion（功能强大）。

## 一、Midjourney：小白首选，5分钟上手

Midjourney是目前最容易上手的AI绘画工具，效果惊艳，学习成本低。

### 如何开始使用？

**步骤1：注册Discord账号**
Midjourney运行在Discord平台上，先注册一个Discord账号。

**步骤2：加入Midjourney服务器**
访问midjourney.com，点击"Join the Beta"，进入Discord服务器。

**步骤3：开始生成图片**
在聊天框输入：`/imagine prompt: 你的描述词`

就这么简单！

### Midjourney基础提示词公式

**万能公式**：
`主体 + 风格 + 环境 + 灯光 + 参数`

**示例**：
`一只可爱的猫咪，卡通风格，坐在窗台上，自然光，ar 16:9`

**拆解**：
- 主体：一只可爱的猫咪
- 风格：卡通风格
- 环境：坐在窗台上
- 灯光：自然光
- 参数：ar 16:9（图片比例）

### 实用提示词库

**风格类**：
- 摄影写实：photorealistic, hyperrealistic
- 插画风格：illustration, digital art
- 水彩画：watercolor painting
- 油画：oil painting
- 3D渲染：3D render, octane render

**光影类**：
- 自然光：natural light
- 黄金时刻：golden hour
- 电影光效：cinematic lighting
- 霓虹灯光：neon lights

**质量类**：
- 高质量：high quality, detailed
- 超高清：8K, ultra detailed
- 大师作品：by ArtStation, by famous artist

### Midjourney进阶技巧

**技巧1：使用图片参考**
上传参考图片，然后输入：`/imagine prompt: [图片链接] 你的描述词`
AI会根据参考图片风格生成新图。

**技巧2：调整权重**
用`::数字`调整重要性：
`猫::2 狗::1`（猫的重要性是狗的2倍）

**技巧3：排除元素**
用`--no`排除不需要的元素：
`一只猫 --no 黑白`（不要黑白猫）

## 二、Stable Diffusion：功能强大，无限可能

Stable Diffusion是开源免费的工具，功能更强大，但需要一定配置。

### 如何部署Stable Diffusion？

**方案1：本地部署（推荐）**
1. 下载安装 Python 和 Git
2. 克隆 Stable Diffusion WebUI
3. 下载模型文件
4. 启动使用

**方案2：云端部署（简单）**
使用 Google Colab 免费算力，搜索"Stable Diffusion Colab"教程。

**方案3：在线平台（省心）**
- Civitai.com
- Leonardo.ai
- SeaArt.ai

### Stable Diffusion核心概念

**1. 模型（Checkpoints）**
模型决定画风，常用模型：
- Realistic Vision：写实照片
- DreamShaper：通用插画
- Anything：动漫风格

**2. 提示词（Prompt）**
正向提示词：你想要的
负向提示词：你不想要的（如：低质量、变形等）

**3. LoRA（微调模型）**
让模型学会特定风格或人物
- 人物LoRA
- 风格LoRA
- 概念LoRA

### Stable Diffusion实用技巧

**技巧1：使用大模型提示词**
提示词网站：
- Prompthero（大量示例）
- LiblibAI（中文提示词库）

**技巧2：控制网（ControlNet）**
精确控制构图、姿势、边缘等，是SD的杀手级功能。

**技巧3：图生图**
用参考图生成新图，保持构图和风格。

## 三、实战案例：从0到1完成一幅作品

让我用Midjourney演示完整流程：

**任务：为公众号文章配一张封面图**

**第1步：确定需求**
- 主题：AI技术
- 风格：科技感
- 色调：蓝色为主

**第2步：编写提示词**
```
A futuristic AI brain glowing with blue light, digital neural network connections, holographic interface, cyberpunk style, high tech, 8K, cinematic lighting --ar 16:9
```

**第3步：生成和迭代**
- 生成4张图片
- 选择最好的那张
- 点击"Vary"按钮微调
- 点击"Upscale"高清放大

**第4步：后期调整**
用Photoshop或其他工具简单调色，完成！

整个过程不到10分钟。

## 四、常见问题解答

**Q：AI绘画会侵权吗？**
A：Midjourney和SD生成的作品版权归属目前还有争议。商业使用建议：
- 使用付费版Midjourney（有商业使用授权）
- 使用开源SD
- 注意不要直接模仿特定艺术家的独特风格

**Q：生成的图片质量不稳定怎么办？**
A：
- 提示词更具体详细
- 使用高质量参考图
- 多生成几次选最好的
- 学习常用提示词组合

**Q：哪个工具更适合新手？**
A：
- 完全新手：Midjourney
- 想深入学习：Stable Diffusion
- 不想折腾：在线平台（Leonardo.ai）
