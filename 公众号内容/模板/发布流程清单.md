# 公众号发布流程清单

## 发布前检查

### 内容质量检查
- [ ] 标题吸引人且符合字数限制（不超过25字）
- [ ] 开头有痛点或热点引入
- [ ] 正文结构清晰（3-5个小节）
- [ ] 每个观点有案例或数据支撑
- [ ] 语言通俗易懂，避免专业术语堆砌
- [ ] 段落长度适中（不超过5行）
- [ ] 重点内容已加粗
- [ ] 结尾有总结和行动建议
- [ ] 文章字数合理（1500-2500字）

### 排版检查
- [ ] 段间距统一（空一行）
- [ ] 小标题格式一致
- [ ] 图片清晰且有说明
- [ ] emoji使用适度（不滥用）
- [ ] 引用块使用正确
- [ ] 手机预览效果良好

### 互动元素
- [ ] 引导点赞/在看/转发
- [ ] 设置互动问题
- [ ] 预告下期内容
- [ ] 加上相关话题标签

### SEO优化
- [ ] 标题包含核心关键词
- [ ] 文章摘要（可选）
- [ ] 合适的分类标签
- [ ] 原文声明（如果需要）

## 发布步骤

### 第一步：公众号后台编辑
1. 登录微信公众平台
2. 新建图文消息
3. 复制markdown内容
4. 调整排版格式
5. 插入配图（如果需要）

### 第二步：预览检查
1. 发送预览到手机
2. 检查手机显示效果
3. 修改格式问题
4. 确认无误后准备发送

### 第三步：选择发布时间
- **推荐时间段**：
  - 早高峰：7:00-9:00
  - 午休：12:00-13:30
  - 晚高峰：18:00-21:00
- **建议**：固定时间发布，培养阅读习惯

### 第四步：正式发布
1. 点击"群发"按钮
2. 选择同步到其他平台（可选）
3. 确认发布
4. 保存发布链接

## 发布后运营

### 内容同步
- [ ] 保存到"已发布"目录
- [ ] 更新选题池状态
- [ ] 同步到知乎/头条/小红书等平台

### 数据监控
发布后24小时内关注：
- 阅读量
- 点赞数
- 在看数
- 分享数
- 收藏数
- 新增关注数
- 留言互动

### 互动维护
- [ ] 及时回复留言
- [ ] 精选优质评论
- [ ] 整理读者反馈
- [ ] 收集选题建议

### 复盘总结
每周/每月复盘：
- 哪类文章数据最好？
- 读者在关心什么话题？
- 标题如何优化？
- 发布时间如何调整？

## 文件命名规范

### 草稿文件
格式：`编号-标题.md`
示例：`001-学会这5个提示词技巧.md`

### 已发布文件
格式：`YYYYMMDD-编号-标题.md`
示例：`20250107-001-学会这5个提示词技巧.md`

### 归档建议
- 每月创建一个子文件夹
- 定期清理旧草稿
- 保留高数据文章备选
